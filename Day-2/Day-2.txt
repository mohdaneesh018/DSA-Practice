Question 5: Reverse an Array (Array ko reverse karo without using extra array)

Input: [1, 2, 3, 4, 5];

Output: [5, 4, 3, 2, 1];

Variables:
Do index lenge:
- start → array ke first index par
- end → array ke last index par

Step 1:
start ko 0 se initialize karo
end ko last index se initialize karo

Step 2:
Jab tak start < end ho:
- start aur end index ke elements ko aapas me swap karo

Step 3:
Swap ke baad:
- start ko 1 se aage badhao
- end ko 1 se peeche lao

Step 4:
Jab start >= end ho jaaye:
- loop stop kar do

Final:
Array reverse ho chuka hoga
